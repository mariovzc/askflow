# SuperBlog



Desafio de MakeitReal.

Funcionalidades:

  - Registrarse
  - Ingresar
  - Listar Preguntas
  - Ver Preguntas
  - Crear Preguntas
  - Editar Preguntas
  - Eliminar Preguntas
  - Comentar Preguntas
  - Votar Preguntas
  - Responder comentarios
  - Votar comentarios



### Gemas

* gem 'annotate'
* gem 'sorcery'
* gem 'rails-i18n', '~> 5.0.0'
* gem 'acts_as_votable', '~> 0.11.1'
* gem 'impressionist'
* gem 'friendly_id', '~> 5.1.0'  4.0+

### Installation


```sh
$ cd askflow
$ bundle install
$ rake db:migrate && rake db:seed
$ rails s
```
